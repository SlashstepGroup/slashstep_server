# Slashstep Server
**Slashstep Server** is the backend of Slashstep, an open-source, project and issue tracking software.

## Features
### Issue management
* **Create and manage work:** Keep track of what matters with items.
* **Manage dependencies:** Link items together with connections.

### Problem management
* **Track and log problems:** You can use your client to send problems to Slashstep Server.
* **Send requests for approval:** Admins can respond to requests, and Slashstep Server can automatically run actions for you.

### Customization
* **Add more with third-party apps:** Most endpoints are compatible with apps. Apps can extend Slashstep Server without you having to change the core code.
* **Automate with workflows:** Automatically run actions after certain events without having to host an additional server.
* **Categorize with fields:** All fields are customizable and hidable.

### Security and compliance
* **Audit activity with logs:** Slashstep Server automatically logs endpoint activity.
* **Manage access on all levels:** From the server-level to the field-level, Slashstep Server features multiple levels of access for maximum granularity.
